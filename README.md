# 🚀 2024 Vibe Coding 期末專案：黑暗極簡風數位名片

## 🌟 1. 專題願景 (Vibe)

* **目標對象：** 大學生、對程式有興趣的人、想了解我的人
* **設計風格：** 黑暗極簡風 (Dark Minimalist) + 玻璃擬態效果 (Glassmorphism)
* **核心功能：** 一頁式數位名片，展示個人經歷、專長、GitHub 作品與未來目標

### 設計理念
打造「高級感」的黑暗主題個人網站。深黑色背景搭配紫藍漸層光暈，營造專業且有質感的氛圍。使用玻璃擬態卡片效果和微妙的動畫，讓設計既極簡又不失細節。

---

## 🤖 2. Gemini 協作紀錄 (Prompt Log)

### 關鍵提示詞

> **Prompt：** 「請幫我設計一個黑暗極簡風的數位名片網站。我是大學生，想展示我的經歷、專長、興趣與未來目標。風格要使用深黑色背景、紫藍漸層強調色、玻璃擬態卡片效果，並且要有現代感的動畫效果。另外請把我的 GitHub 專案加入作品集區塊。」

### 問題解決紀錄

* **遇到的問題：** 需要展示 GitHub 上的專案作品
* **AI 的建議：** 透過 GitHub API 抓取公開儲存庫資訊，並設計專案卡片區塊來展示每個作品的名稱、描述和技術標籤

* **遇到的問題：** 技能條動畫在手機上沒有觸發
* **AI 的建議：** 使用 Intersection Observer API 偵測元素可見性，確保動畫正常運作
